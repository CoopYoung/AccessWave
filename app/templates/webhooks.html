Verifying signatures

Each POST request includes an X-AccessWave-Signature header with a sha256=<hex> HMAC-SHA256 digest of the raw request body, signed with your webhook’s secret.

Example verification in Python:

import hashlib, hmac def verify(secret: str, body: bytes, signature: str) -> bool: expected = "sha256=" + hmac.new( secret.encode(), body, digestmod=hashlib.sha256 ).hexdigest() return hmac.compare_digest(expected, signature)

Return a 2xx status code within 10 seconds to acknowledge delivery. Failed deliveries are not retried automatically.
